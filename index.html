Supabase & Vercel

○ verificando…

Configure as credenciais do Supabase para persistência em nuvem e deploy no Vercel. A anon key é pública por design — use-a aqui. A service role key vai só no .env do servidor.

GitHub Actions — Busca automática online

Configure um Personal Access Token do GitHub para que o botão "Executar agora" dispare a busca online (workflow_dispatch), mesmo sem o servidor local.

Gere o token em: GitHub → Settings → Developer settings → Personal access tokens → Fine-grained → permissão Actions: Read and write.

Deploy no Vercel

  1. Na pasta do projeto: vercel --prod
  2. Adicione os secrets no GitHub (repo → Settings → Secrets → Actions): ANTHROPIC_API_KEY, SUPABASE_URL, SUPABASE_KEY, ADZUNA_APP_ID, ADZUNA_APP_KEY, JOOBLE_API_KEY
  3. O workflow roda automaticamente seg–sex às 08:00 (BRT)
